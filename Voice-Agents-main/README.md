# ***Starpower Voice Agents***

This repo is designed for a new approach for UX, making agent use hands free & creating a seamless conversation with backend agents doing the heavy lifting. Were basically giving a voice model its own helpful assistant. I believe socialism in agentic systems are the future.

This setup has a full app already created as a demo for what is possible for web search, file management & an interactive reading loop. 

- Hands free UI popups
- Backend agent for writing documents & web searches
- Realtime voice mode
